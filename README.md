# Facchini.EPI — Sistema Inteligente de Monitoramento de EPIs

Este projeto apresenta uma proposta inovadora e tecnológica para a gestão, rastreabilidade e monitoramento automático de Equipamentos de Proteção Individual (EPIs) em ambientes fabris de grande porte, com foco na realidade industrial da **Facchini S.A.**

Desenvolvido com carinho e esforço pelos alunos da turma **TDS 2025** para demonstrar o potencial de Visão Computacional aplicada à Segurança do Trabalho.

## 🚀 Funcionalidades Demonstradas
* **Dados Reais**: Painel estatístico nacional de acidentes e riscos associados à ausência de EPIs.
* **Demonstrações Interativas**: Simulação de alertas inteligentes e logs de reação em três cenários industriais críticos:
  1. Linha de Soldagem
  2. Setor de Montagem
  3. Cabine de Pintura
* **PowerPoint Integrado**: Gerador de slides automatizado que gera o arquivo de apresentação técnica final `.pptx` de forma dinâmica.

## 💻 Como Rodar Localmente

1. Instale as dependências de geração de slides:
   ```bash
   npm install
   ```

2. Gere a apresentação PPTX atualizada:
   ```bash
   node index.js
   ```

3. Abra o arquivo `index.html` em qualquer navegador moderno para interagir com os slides e simulações de câmeras ao vivo.
